表单元素

利用title属性在右边增加文本

示例:
<input type="checkbox" class="bui-choose" name="sex2" value="1" title="羽毛球">

利用label标签来自定义

单选框 bui-radio

示例:
<input type="radio" class="bui-radio" name="sex" value="1">

多选框 bui-checkbox

示例:
<input type="checkbox" class="bui-checkbox" name="sex2" value="1">

单选 bui-choose

示例:
<input type="radio" class="bui-choose" name="choose" value="1">

多选 bui-choose

示例:
<input type="checkbox" class="bui-choose" name="choose" value="1">

自定义多选单选 bui-check

示例:
<div class="bui-check"> <input id="money1" type="checkbox"> <label for="money1">50元</label> </div>

喜欢 bui-like

示例:
<input type="checkbox" class="bui-like" name="like" value="2">

切换器 bui-switch

示例:
<input type="checkbox" class="bui-switch" name="interest" value="1">开启

切换器 bui-switch-text

示例:
<input type="checkbox" class="bui-switch-text" name="interest2" value="1" uncheck="OFF" check="ON">

滑动条 bui-range

示例:
<input type="range" class="bui-range" min="0" max="255" value="200">

系统自带日期 bui-date

示例:
<input type="date" class="bui-date" >

input框

示例:
<div class="bui-input"> <i class="icon-search"></i> <input type="text" value="" placeholder="请输入关键字"> <i class="icon-remove"></i> </div>

搜索条

示例:
<div class="bui-searchbar"> <div class="bui-input"> <i class="icon-search"></i> <input type="text" value="" placeholder="请输入关键字"><i class="icon-remove"></i> </div> </div>

搜索条-带文字

搜索
示例:
<div class="bui-searchbar bui-box"> <div class="span1"> <div class="bui-input"> <i class="icon-search"></i> <input type="text" value="" placeholder="请输入关键字"><i class="icon-remove"></i> </div> </div> <div class="btn-search">搜索</div> </div>

搜索条-带条件

条件
  • 列表1
  • 列表2
  • 列表3
示例:
<div class="bui-searchbar"> <div class="bui-input"> <div class="bui-dropdown" style="width:2rem;"> <div class="bui-btn bui-box"> <div class="span1">条件</div> <i class="icon-dropdown"></i> </div> <ul class="bui-list bui-menu-bottom"> <!-- 默认激活第一个 --> <li class="bui-btn" value="11"> 列表1 </li> <li class="bui-btn" value="22"> 列表2 </li> <li class="bui-btn" value="33"> 列表3 </li> </ul> </div> <input type="text" placeholder="请输入关键字搜索" value=""> <i class="icon-search"></i> </div> </div>

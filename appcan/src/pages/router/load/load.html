跳转新页面-router.load

load 页面跳转,更多参数用法请参考 bui.router 的 api

用法: router.load(obj);

**注意** 如果要保持跟多页方式一致,则使用 bui.load(obj)

字段名 类型 必须
obj object 是
obj.url string 是
obj.param object 否

普通跳转 router.load

示例:
// 页面跳转 router.load({ url: "pages/page2/page2.html" }); 注意:单页的页面跳转可以保持跟多页开发方式一致, 也就是你也可以使用 bui.load, 并且单页的跳转只需像index.js 一样,初始化一次就可以了

页面传参 router.load

示例:
// 页面跳转并且传参数给 pages/page2/page2 param 为要传过去的参数 router.load({ url: "pages/page2/page2.html",param: {"name":"page2"} });

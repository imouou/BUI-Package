BUI单页路由
单页路由详细说明
  • 1. 路由初始化以后,第一次打开首页,便会加载 main 模块, 每个模块需要通过loader.define定义
  • 2. 默认main是在pages/main/目录下, 一个main模块,包含 main.html(模板), main.js (业务逻辑)
  • 3. 一个js只能用loader.define定义一个匿名模块,后面的会覆盖前面的,具体使用请查看API
  • 4. 页面跳转刷新返回传参等,推荐保持跟多页的开发方式保持一致.

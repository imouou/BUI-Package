loader.require

模块默认已经初始化给 window.loader

用法: loader.require(depend,callback);

字段名 类型
depend array
callback function
用法1: 加载匿名模块示例:
js: // 加载匿名模块, 匿名模块默认以脚本路径为名,无需.js loader.require(["pages/page2/page2"],function(page2){ // page2 == {page:"page2"} })
用法2: 加载指定模块示例:
js: loader.require(["page2"],function(page2){ // page2 == {page:"page2"} })
